# SolMarket Brand Guidelines

## Overview
SolMarket is a premium decentralized marketplace built on the Solana blockchain. Our brand identity combines cutting-edge technology with elegant design, creating a sophisticated and trustworthy platform for digital commerce.

## Color Palette

### Primary Colors
- **Neon Purple**: `hsl(280, 100%, 60%)` - Primary brand color
- **Electric Blue**: `hsl(220, 100%, 60%)` - Secondary accent
- **Cyber Pink**: `hsl(330, 100%, 65%)` - Tertiary accent

### Metallic Gradients
- **Gold Metallic**: `hsl(45, 85%, 55%)` - Premium features
- **Silver Metallic**: `hsl(210, 10%, 75%)` - Professional elements
- **Bronze Metallic**: `hsl(30, 65%, 45%)` - Legacy features

### Semantic Colors
- **Success**: `hsl(160, 85%, 45%)` - Completed transactions
- **Warning**: `hsl(45, 85%, 55%)` - Pending actions
- **Error**: `hsl(0, 85%, 55%)` - Failed transactions
- **Info**: `hsl(220, 85%, 55%)` - Informational content

### Monochromatic Variants
Each primary color has 10 variants (50-900) for consistent hierarchy and depth.

## Typography

### Primary Font
- **Geist Sans** - Modern, clean sans-serif for UI elements
- **Geist Mono** - Monospace for code and technical data

### Typography Hierarchy
- **Hero**: 5xl-8xl, gradient text, bold weight
- **Headings**: 2xl-4xl, gradient or solid, semibold
- **Body**: base-lg, regular weight
- **Small**: xs-sm, regular weight

### Text Effects
- **Gradient Text**: Vibrant gradient for emphasis
- **Glitch Text**: Digital aesthetic for headings
- **Neon Glow**: Interactive text elements

## Visual Effects

### Animations
- **Float**: Gentle vertical movement (6s duration)
- **Pulse Glow**: Rhythmic glow effect
- **Shimmer**: Horizontal light sweep
- **Liquid Morph**: Organic shape transitions
- **3D Rotation**: Spatial depth effects

### Micro-interactions
- **Magnetic Cursor**: Elements follow cursor movement
- **Ripple Effects**: Click feedback
- **Spring Bounce**: Elastic transitions
- **Elastic Scale**: Dynamic resizing

### Glassmorphism
- **Glass**: 5% background opacity
- **Glass Strong**: 10% background opacity
- **Glass Tint**: Color-tinted variants

## Iconography

### Custom Icons
- **SolMarket Logo**: Hexagonal blockchain symbol
- **Solana Icon**: Official Solana branding
- **Security Shield**: Trust indicators
- **Transaction Flow**: Data visualization
- **Marketplace**: Store representation
- **Wallet**: Digital assets

- **Line weight**: 2px stroke

## Layout Principles

### Grid System
- **Base grid**: 12-column responsive grid
- **Spacing**: 8px base unit
- **Containers**: Max-width constraints
- **Breakpoints**: sm (640px), md (768px), lg (1024px), xl (1280px)

### Composition
- **Golden ratio**: 1.618 proportions
- **Visual hierarchy**: Size, color, placement
- **White space**: Generous breathing room
- **Depth layers**: Multiple z-index levels

## Component Guidelines

### Cards
- **Border radius**: 12px
- **Shadow**: Multi-layer glow effects
- **Hover state**: Scale + glow + shimmer
- **3D effects**: Perspective transforms

### Buttons
- **Height**: 56px (large), 40px (medium), 32px (small)
- **Border radius**: 8px
- **States**: Default, hover, active, disabled
- **Liquid morph**: Border shape transitions

### Forms
- **Inputs**: Glassmorphism with glow
- **Labels**: Enhanced readability
- **Validation**: Semantic color indicators
- **Focus states**: Animated outline

## Accessibility Standards

### Contrast Ratios

### Motion Preferences
